DrupAck
=======

A simple bash shell script to ack Drupal code for patterns known to cause security issues.

## Requirements

ack

## Usage

`drupack [directory]`

### Whats the name?

Drupal + Ack = drupack

I'm open to ideas.

### Inspiration

This project was inspired by a section of the book Cracking Drupal by @greggles.
